# Social Site Project

This is a simple frontend project of a social media website.

## Features
- Navigation bar (Home, Profile, Friends, Messages, Notifications)
- Search bar functionality
- Logout button
- Multiple pages (Home, Profile, Friends, Messages, Notifications)
- Admin pages (Login, Dashboard, User Management, Post Management, etc.)

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Open the project folder
2. Double click on index.html
3. Use the navigation bar to explore pages
